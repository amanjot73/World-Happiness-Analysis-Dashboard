This repository contains a **Power BI data analytics project** focused on analyzing global happiness trends using the **World Happiness Report dataset**.  
The project applies data preparation, DAX calculations, data modeling, and interactive visualizations to derive meaningful insights about happiness across countries, regions, and time.

---

## ğŸ“Œ Project Objectives

The dashboard was designed to achieve the following objectives:

1. **Compare Happiness Across Countries**  
   - Identify the happiest and least happy countries  
   - Compare happiness scores globally using rankings and KPIs  

2. **Analyze Happiness Trends Over Time**  
   - Study year-wise changes in happiness  
   - Identify improving and declining trends  

3. **Factor Impact Analysis**  
   - Analyze how GDP per capita, social support, health, freedom, generosity, and corruption influence happiness  

4. **Regional Comparison & Insights**  
   - Compare happiness levels across regions  
   - Identify geographic patterns and disparities  

5. **Ranking & Inequality Analysis**  
   - Compare Top 10 vs Bottom 10 countries  
   - Measure persistent happiness gaps  

6. **Advanced Insight & Decision Analysis**  
   - Use Key Influencers and Decomposition Tree visuals  
   - Identify drivers behind happiness variations
